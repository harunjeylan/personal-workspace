# Personal Workspace

This directory serves as a personal workspace for storing projects, specialized agents, skills, tools, and workflows.

## Directory Structure

- `agents/` - Custom specialized agents
- `skills/` - Custom skills
- `tools/` - Custom tools
- `workflows/` - Workflow configurations
- `projects/` - Personal projects
- `config/` - Configuration files

## Getting Started

1. Add custom agents to the `agents/` directory
2. Create skills in the `skills/` directory (following the SKILL.md format)
3. Add tools to the `tools/` directory
4. Store workflows in the `workflows/` directory
5. Create projects in the `projects/` directory

## Available Skills

- **Code Review Skill** (`skills/code-review-skill/`): Comprehensive code reviews with security, performance, and best practices focus.
- **Commit Message Skill** (`skills/commit-message-skill/`): Generates conventional commit messages based on code changes.
- **Documentation Skill** (`skills/documentation-skill/`): Helps generate and maintain documentation for code and projects.

## Git

The repository is initialized and ready for version control. Common temporary files are excluded via `.gitignore`.
